# 📊 Sales Intelligence Hub

## 📌 Project Description

Sales Intelligence Hub is a **branch-based sales management system** built using **Streamlit and MySQL**.
It helps businesses track sales, manage split payments, monitor pending amounts, and gain insights through a centralized dashboard.

---

## 🚀 Features

* 🔐 User Login System (Admin & Super Admin)
* 🏢 Branch-wise Sales Tracking
* 💰 Split Payment Management
* 🔄 Automatic Calculation using SQL Triggers
* 📊 KPI Dashboard (Sales, Received, Pending)
* 📈 Sales Analysis & Reporting
* 🗄️ MySQL Database Integration

---

## 🛠️ Technologies Used

* Python
* Streamlit
* MySQL
* Pandas

```

```

```

```

### Step 3: Run Application

